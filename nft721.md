何为nft？

```
token是按照金融交易，制定交易法律
nft是按照艺术品，制定显示法律

token都是一个模子的交易律法，同质化代币
nft是可以制定的，个性化的显示律法，非同质化合约，非代币也不具备交易性质
```

```
例如：颜色nft，把#fff颜色数据上链存储。这就具备了区块链特性
目前钱包没有展示的功能，但数据是在合约的，这里可以通过中心化服务器替代显示
把合约存储的nft数据读取出来展示，做到了用户永久持有，虽然目前没有很区块链色彩的展示
nft遵循721规范合约。一个nft就意味着发行量+1，这和token发行量不一样【token发行量一开始固定，nft不断发行】
```
